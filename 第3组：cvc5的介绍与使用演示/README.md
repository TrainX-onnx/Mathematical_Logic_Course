# cvc5
## 简介

cvc5 是一个高性能的 SMT（可满足性模块理论）求解器，广泛用于逻辑推理、程序验证和自动化问题解决.

## 参考网址

https://github.com/cvc5/cvc5


cvc5 online（SMT-LIB 标准）
